# Vakratund-Store-Analysis (Interactive Dashboard Creation Using MS Excel)
## 🎯Project Objective
The objective of this project is to analyze and visualize sales performance, customer behavior, and order trends of Vakratund Store using an interactive dashboard. The goal is to transform raw sales data into meaningful insights that support data-driven decision-making. Overall, the project helps in improving business strategies, increasing sales efficiency, and enhancing customer targeting through data insights.

## 📊Dataset Used
- <a href="https://github.com/anilsakhare-gthb/data-analysis-dashboard/blob/main/vakratund-store-analysis.csv">Vakratund Store Dataset</a>

## ❓Questions (Dashboard Requirements)
- In which month are both sales amount and order count at their peak.
- Is there any month where sales increased but order count decreased (or vice versa)? What could be the reason.
- Which gender contributes more to total sales, and what is their percentage share.
- What is the combined percentage of problematic orders (Cancelled + Returned + Refunded).
- Which state generates the highest sales, and how does it compare to the lowest among the top 5 states.
- Which age group has the highest order contribution, and how does it differ between men and women.
- Which sales channel contributes the highest share of orders.
- What is the percentage difference between the top two sales channels.
