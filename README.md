# Vakratund-Store-Analysis (Interactive Dashboard Creation Using MS Excel)
## 🎯Project Objective
The objective of this project is to analyze and visualize sales performance, customer behavior, and order trends of Vakratund Store using an interactive dashboard. The goal is to transform raw sales data into meaningful insights that support data-driven decision-making. Overall, the project helps in improving business strategies, increasing sales efficiency, and enhancing customer targeting through data insights.
